OUR MISSION

We believe that a community is as strong as its members. We want to create societies which are so self-efficient that no matter the circumstances it will stand strong like a rock.

Through Padosi- the community app, everyone in a community will have an option to ask help from another member and can also provide help for safely delivering their Groceries, dairy products, medicines, fruits and vegetables.

How the app will work


  • The user will be asked to make a shopping list based on categories. Once the list is made, the user will tell us if they are going out to buy the products in the list or want help with their delivery.

  • Based on the option selected, there will be two lists maintained

  1. Going out

  2. look out


  • If the user selects Look out, other users within a certain set radius will be able to see their request and accept the same.

  • If the user selects Going out, other users within a certain set radius will be able to see the same and send you a personal request.

  • The users going out do not have to necessarily make the shopping list, they can go to "Going out" from home page, Go to "Open Requests" and directly accept requests.


Payment

The application does not have a wallet but will show what all payment portal each user has for better clarity. The users can then pay accordingly, off the platform.
